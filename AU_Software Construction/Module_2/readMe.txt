M2: Introduction
Introduction
In this module, we will go over the software development process, decision structures, organizing code with methods, and the basics of classes in C#. We will also discuss your course project requirements!

 

Learning Objectives
By successfully completing the module, you will be able to:

Identify different processes used to build software
Organize a software project into three layers: presentation, logic, and data
Use control structures (decisions and loops) in a C# application
Modularize your code into methods
Create classes and identify their members correctly (fields, methods, properties)
Identify potential classes for your course project based on the project description
