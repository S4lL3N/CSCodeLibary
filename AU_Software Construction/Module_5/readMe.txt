M5: Introduction
Introduction
In this module you will learn different ways you can work with data in your Windows Forms Applications. You will learn how to use both files and databases to store and retrieve data! You will learn the basic concepts of relational databases, as well as how to use Windows Forms data binding controls to connect to, display, and edit data in a database. For your course project, you will use what you learned in this module to implement additional functionality in your web browser!


 

Learning Objectives
By successfully completing the module, you will be able to:

Use files to save and retrieve data in a Windows Forms Application
Design an application data model using relational database concepts
Bind data controls to data sources in your Windows Forms Application
Use databases to store and retrieve data
