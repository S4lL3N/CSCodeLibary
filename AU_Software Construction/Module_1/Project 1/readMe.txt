M1: Introduction
Introduction
From installing Visual Studio, to introducing the C# programming language, to exploring the basics in managing your Visual Studio development environment, this module contains everything you need to get started with Visual C# and create your first Windows Forms application!

 

Learning Objectives
By successfully completing the module, you will be able to:

Install Visual Studio
Create projects and debug your code using Visual Studio
Write basic programs using the C# programming language
Build a simple Windows Forms application
Manage your source code using version control (Git)
