M3: Introduction
Introduction
In this module you will learn to build more robust applications using unit testing with NUnit as well as exception handling. You will be introduced to some Windows Forms controls and some of their commonly used properties and events. You will start designing your user interface for your course project as well!

 

Learning Objectives
By successfully completing the module, you will be able to:

Manage NuGet packages in your project
Use NUnit for unit testing your C# applications
Use exception handling in your C# applications
Use common data structures (Lists and Arrays) to solve problems
