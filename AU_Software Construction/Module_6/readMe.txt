Introduction
In this module you will learn about additional Windows Forms controls that can be used in your Windows Forms Applications. You will also learn how to package and deploy your Windows Forms Application using ClickOnce technology as your course project implementation nears completion! In addition, we will cover the topics of Inheritance and Polymorphism in Visual C#.

 

Learning Objectives
By successfully completing the module, you will be able to:

Use Additional Windows Forms Controls in your Windows Forms Applications such as the StatusStrip, ProgressBar, and Timer controls.
Use Inheritance and Polymorphism effectively in your Visual C# applications.
Package and publish your 3-tier Windows Forms Application using ClickOnce technology.
