Introduction
In this module you will be introduced to some advanced GUI features that can be used to enhance your Windows Forms Application. You will learn how to create a multi-form application and build your own user controls. You will use these features to implement additional functionality in your course project.

 

Learning Objectives
By successfully completing the module, you will be able to:

Add containers, selection controls, and keyboard shortcuts to your Windows Forms Applicaiton
Create multi-form Windows Forms Applications with both modal and modeless forms
Build your own custom user controls and use them in your applications
